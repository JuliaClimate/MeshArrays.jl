MeshArrays.jl

MeshArrays.jl defines an array type that can contain / organize / distribute collections of inter-connected arrays as generally done in climate models (see Earth Grids). MeshArrays.jl's data structures can be used to simulate and analyze key variables of the climate system such as global transports and particle displacements.

Note

Global Ocean Notebooks, IndividualDisplacements.jl, and MITgcmUtils.jl illustrate various use cases for example.
